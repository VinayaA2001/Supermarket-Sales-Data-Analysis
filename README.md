# Supermarket Sales Data Analysis

# Overview
This project performs exploratory data analysis (EDA) on supermarket sales data to understand
sales trends, customer behavior, and product performance across branches and cities.

# Key Analysis Performed
- Branch-wise and city-wise sales analysis
- Product line performance evaluation
- Customer type and gender-based sales insights
- Payment method distribution
- Monthly and hourly sales trend analysis
- Heatmap visualization using pivot tables

# Key Insights
- Certain branches consistently outperform others in total sales.
- Specific product lines generate higher revenue across locations.
- Member customers contribute more to overall revenue.
- Sales peak during specific hours of the day, indicating optimal business timings.

# Conclusion
The analysis provides actionable business insights that can help improve inventory planning,
customer targeting, and operational decision-making.
